Buttons

There is nothing complicated going on here. Buttons can simply be applied with the .btn class. The aim is to keep a flat design that fits the colorscheme.

<button class="btn">Press Me</button>
<button class="btn btn-cyan">Cyan</button>

Cards can be used to display any kind of information. You can use to it display products, blog posts and more.

Forms

On any given website, you will probably need to use a form somewhere. So far Gruv CSS only has some simple styling for labels and inputs.

Input fields

The input tag is field used in forms to receive user data. In order to achieve the proper styling, add the .form-group class inside the form. This will stack the elements inside of it.
