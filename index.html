About

Gruv CSS is a minimal and responsive CSS framework based on the popular colorscheme, gruvbox. Gruv CSS started out as a personal project because I relied too much on 3rd party css frameworks like Bootstrap, Bulma and Materialize. I wanted something that was more in line with my personal taste when it comes to design and color use.

The Gruvbox colorscheme has been my all time favorite colorscheme. I use for my browser, filemanager, terminal and wherever I can.
